186 """
187 Array API compatible wrapper for :py:func:`np.matrix_rank <numpy.matrix_rank>`.
188  
189 See its docstring for more information.
190 """
191 # Note: this is different from np.linalg.matrix_rank, which supports 1
192 # dimensional arrays.
